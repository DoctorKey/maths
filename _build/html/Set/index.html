集合论¶

在数学不断地发展过程中，集合论渐渐成为了数学的基础。

在这一部分中，我们谈谈集合论的相关东西，为之后的拓扑，时变，泛函做个准备。当然，你也可以直接阅读其他部分，遇到疑问的地方再回来查阅集合论的相关内容。

Note

本部分并非一个讲解集合论的教程，对集合论的相关部分只是简单介绍，如想仔细学习集合论，建议学习相关的专业书籍。
